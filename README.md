# React MobX Shop

## Git clone

```
git clone https://github.com/andreyskorchenko/reactmobxshop.git
```

## Change directory

```
cd reactmobxshop
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```
